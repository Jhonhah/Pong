A simple, classic Pong game built using Python’s built‑in turtle graphics module:
(1st Capstone project of 100 Days of Code: The Complete Python Bootcamp) 
Features:
- Two‑player paddle controls
- Ball movement with bouncing and collision detection
- Score tracking

Controls
Function       Player 1(left)    Player 2(right)    
Paddle UP      W                 Up
Paddle Down    S                 Down
